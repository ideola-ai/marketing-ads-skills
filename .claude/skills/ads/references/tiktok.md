# TikTok Ads Audit Guide

## Ad Formats

### 1. In-Feed Ads
**Length:** 5-60 seconds
**Placement:** User's "For You" feed
**Key Checks:**
- 2-second video view rate (VTR)
- 6-second video view rate
- Video completion rate
- CTR

**Benchmarks:**
- VTR (2s): 20-30%
- VTR (6s): 10-20%
- Completion rate: 15-25%
- CTR: 1.0-2.5%
- CPC: $0.50-1.50

### 2. Spark Ads
**Type:** Amplified organic content
**Key Checks:**
- Organic vs. paid performance comparison
- Engagement rate (likes, comments, shares)
- Sound-on usage
- Creator performance

**Benchmarks:**
- VTR (2s): 25-40%
- Engagement rate: 3-8%
- CTR: 1.5-3.0%

### 3. TopView
**Type:** Full-screen takeover (3-60s)
**Key Checks:**
- View rate
- Completion rate
- Brand lift
- Cost per thousand (CPM)

**Benchmarks:**
- View rate: 40-60%
- Completion rate: 50-70%
- CPM: $50-100

### 4. Brand Takeover
**Type:** First impression of the day
**Key Checks:**
- Daily reach
- CTR
- Brand recall

**Benchmarks:**
- CTR: 5-8%
- CPM: $80-150

### 5. Collection Ads / Shop Ads
**Type:** E-commerce focused
**Key Checks:**
- Product click rate
- Add to cart rate
- Conversion rate
- ROAS

**Benchmarks:**
- Product CTR: 2-5%
- Conversion rate: 2-4%
- ROAS: 1.5-3.0x

## TikTok-Specific Metrics

| Metric | Good | Needs Help |
|--------|------|------------|
| 2s VTR | 25%+ | <15% |

## Creative Best Practices

### Video Structure (3-Second Framework)

| Time | Element | Purpose |
|------|---------|---------|
| 0-0.5s | Visual hook | Stop scroll |
| 0.5-3s | Context/setup | Establish what & why |
| 3-15s | Content delivery | Value/entertainment |
| 15-60s | CTA extension | Optional, for longer videos |

### Hook Formulas That Work
- "POV: [relatable situation]"
- "Stop doing [X], try [Y] instead"
- "The secret to [benefit]..."
- "I tried [X] for [time] and..."
- "This [product] changed [aspect]..."

### Sound Strategy
- **Trending audio:** Check TikTok Creative Center for trending sounds
- **Brand audio:** Custom branded music
- **Voiceover:** Clear, energetic narration
- **Music licensing:** Ensure commercial use rights

### Visual Requirements
- **Aspect ratio:** 9:16 (vertical only)
- **Resolution:** 720x1280 minimum
- **Text overlay:** Keep under 20% of screen
- **Safe zones:** Avoid UI overlap (bottom UI, top captions)

## Targeting Options

### Audience Types
| Type | Description | Best For |
|------|-------------|----------|
| Broad | No targeting | Discovery, learning |
| Interest | Based on content engagement | Awareness |
| Behavior | In-app actions | Consideration |
| Custom | Upload customer list | Retargeting |
| Lookalike | Similar to converters | Acquisition |

## Quick Audit Checklist

- [ ] Review 2s and 6s VTR
- [ ] Check completion rate
- [ ] Analyze hook performance (first 3 seconds)
- [ ] Verify vertical format (9:16)
- [ ] Check sound-on rate
- [ ] Review trending audio usage
- [ ] Compare Spark vs. regular in-feed performance
- [ ] Check creative diversity (avoid repetitive ads)
- [ ] Verify pixel events are firing
- [ ] Review placement targeting (Pangle vs. TikTok)

## TikTok Creative Diversity Score

Calculate diversity by counting unique creative elements:

| Element | Count | Score |
|---------|-------|-------|
| Video angles | 5+ different | High |
| Music tracks | 3+ different | High |
| Hooks/openings | 4+ different | High |
| CTAs | 3+ variations | Medium |
| Creators | 3+ different | High |

**Low diversity:** Same creative running >1 week without change

## Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Low 2s VTR | Improve opening hook, add motion |
| Low completion | Shorten video, improve pacing |
| Low engagement | Use trending audio, ask questions |
| Not delivering | Expand targeting, check approval |
| High frequency | Refresh creatives weekly |
| Low sound-on rate | Use trending sounds, better audio |
| Pixel not firing | Verify installation, test events |

## Creative Fatigue Signals

| Signal | Threshold | Action |
|--------|-----------|--------|
| Frequency | >4 per user | Pause ad, refresh creative |
| Declining CTR | -30% from peak | New creative needed |
| Rising CPA | +50% from baseline | Test new angles |
| Comment sentiment | Negative feedback | Address issues, new creative |
