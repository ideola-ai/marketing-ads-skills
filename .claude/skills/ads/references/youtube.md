# YouTube Ads Audit Guide

## Ad Formats

### 1. Skippable In-Stream Ads
**Length:** 15 seconds - 3 minutes (can be skipped after 5)
**Key Checks:**
- View rate (25%, 50%, 75%, 100%)
- Average CPV (cost per view)
- Skip rate analysis (when do viewers drop?)
- Hook effectiveness (first 5 seconds)

**Benchmarks:**
- View rate (30s): 25-35%
- CPV: $0.10-0.30
- Completion rate: 30-50%

### 2. Non-Skippable In-Stream Ads
**Length:** 15 seconds (maximum)
**Key Checks:**
- Completion rate (should be 90%+)
- Brand lift metrics
- Click-through rate

**Benchmarks:**
- Completion rate: 90%+
- CPV: $0.15-0.40
- CTR: 1-2%

### 3. Bumper Ads
**Length:** 6 seconds (non-skippable)
**Key Checks:**
- Completion rate
- Brand awareness lift
- Recall rate

**Benchmarks:**
- Completion rate: 80%+
- CPV: $0.05-0.15

### 4. YouTube Shorts Ads
**Length:** Up to 60 seconds
**Key Checks:**
- Video view rate
- Engagement rate (likes, comments)
- Swipe-up/CTA clicks

**Benchmarks:**
- View rate: 15-25%
- CPV: $0.05-0.15

### 5. Discovery Ads
**Placement:** Homepage, search results, watch page
**Key Checks:**
- Thumbnail effectiveness
- CTR by placement
- Cost per click

**Benchmarks:**
- CTR: 1-3%
- CPC: $0.20-0.50

## YouTube-Specific Metrics

| Metric | Good | Needs Help |
|--------|------|------------|
| View Rate (30s) | 30%+ | <20% |
| Average CPV | <$0.20 | >$0.35 |
| Completion Rate | 40%+ | <25% |
| CTR (Discovery) | 2%+ | <1% |
| Engagement Rate | 5%+ | <2% |
| Audio ON Rate | 80%+ | <60% |

## Creative Best Practices

### Hook (First 5 Seconds)
- State the problem or value immediately
- Use movement/visual changes
- Include brand early
- Avoid generic intros

### Video Structure
| Element | Timing | Purpose |
|---------|--------|---------|
| Hook | 0-5s | Grab attention |
| Problem | 5-15s | Build need |
| Solution | 15-45s | Present product |
| Proof | 45-55s | Social proof |
| CTA | 55-60s | Clear action |

### Audio Strategy
- Music: Match brand tone, consider trending sounds
- Voiceover: Professional vs. authentic
- Sound effects: Use for emphasis

## Targeting Considerations

### Audiences
- **In-Market:** High intent, good for consideration
- **Affinity:** Interest-based, good for awareness
- **Custom:** Upload your customer list
- **Similar:** Lookalike from converters
- **Keywords:** Videos/topics your audience watches

### Placements
- **In-Stream:** Before/during video
- **Video Discovery:** Thumbnail-based
- **Shorts Feed:** Between Shorts

## Quick Audit Checklist

- [ ] Review view rate by ad format
- [ ] Analyze audience retention graph (drop-off points)
- [ ] Check CPV efficiency
- [ ] Review hook performance (first 5s)
- [ ] Verify CTA clarity and timing
- [ ] Compare skippable vs. non-skippable performance
- [ ] Check audio-on rate
- [ ] Review targeting by audience segment
- [ ] Test thumbnail effectiveness (discovery ads)

## Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Low view rate | Improve hook, test new creative |
| High skip rate | Shorten intro, add value earlier |
| Low completion | Reduce length, improve pacing |
| Low CTR (discovery) | Test new thumbnails |
| High CPV | Refine targeting, improve creative quality |
| Low audio-on rate | Add captions, make audio compelling |
