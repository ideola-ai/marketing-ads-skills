# Ads Audit Checklist

Use this checklist to systematically audit any ad campaign.

## 1. Account Structure

### Campaign Level
- [ ] Campaign objective matches goal
- [ ] Budget is sufficient for learning phase (min 50x target CPA)
- [ ] Bid strategy aligns with objective
- [ ] Campaign duration is appropriate
- [ ] Date range covers meaningful period

### Ad Set Level
- [ ] Targeting is not too narrow (<100K) or too broad (>10M)
- [ ] No overlapping audiences between ad sets
- [ ] Placement optimization is appropriate
- [ ] Budget split makes sense
- [ ] Schedule/dayparting is configured

### Ad Level
- [ ] Multiple creatives per ad set (3-5 minimum)
- [ ] Creative format matches placement
- [ ] Copy has clear hook and CTA
- [ ] Landing page link works
- [ ] Tracking pixel is firing

## 2. Performance Analysis

### Funnel Metrics
- [ ] Impressions are being delivered
- [ ] CTR is within acceptable range
- [ ] Landing page view rate is healthy
- [ ] Conversion rate is tracked
- [ ] Cost per result is calculated

### Efficiency Metrics
- [ ] ROAS is calculated and benchmarked
- [ ] CPA compared to target
- [ ] Budget utilization is >80%
- [ ] Frequency is monitored (not fatigued)

### Quality Metrics
- [ ] Relevance score (Meta) / Quality score (Google)
- [ ] Delivery category (learning, limited, etc.)
- [ ] Disapproval rate
- [ ] Negative feedback rate

## 3. Creative Audit

### Visual Elements
- [ ] Image/video quality is high
- [ ] Brand consistency maintained
- [ ] Product is clearly visible
- [ ] Text overlay is readable (<20% rule for Meta)
- [ ] Mobile-first design

### Copy Elements
- [ ] Strong hook in first 2 seconds/lines
- [ ] Clear value proposition
- [ ] Social proof included
- [ ] Strong CTA
- [ ] Hashtags relevant (not excessive)

### Creative Fatigue
- [ ] Frequency monitored per creative
- [ ] CTR trend analyzed (declining?)
- [ ] Fresh creative in rotation
- [ ] A/B tests documented

## 4. Targeting Audit

### Audience Setup
- [ ] Target audience matches customer profile
- [ ] Exclusions properly configured
- [ ] Retargeting lists are fresh
- [ ] Lookalike audiences are optimized

### Platform-Specific Targeting
- [ ] Meta: Interests, behaviors, demographics
- [ ] Google: Keywords, match types, audiences
- [ ] TikTok: Interests, behaviors, devices
- [ ] LinkedIn: Job titles, industries, skills

## 5. Technical Audit

### Tracking Setup
- [ ] Base pixel installed correctly
- [ ] Conversion events are firing
- [ ] Value tracking is accurate
- [ ] CAPI (Meta) or enhanced conversions (Google) enabled
- [ ] Domain verification complete

### Landing Page
- [ ] Page loads in <3 seconds
- [ ] Mobile-optimized
- [ ] CTA above fold
- [ ] Form/testimonial/social proof present
- [ ] No broken links or errors
- [ ] Privacy policy & terms present

## 6. Budget & Bidding

### Budget Configuration
- [ ] Daily budget sufficient for delivery
- [ ] Budget allocation aligns with performance
- [ ] Seasonality considered
- [ ] Pacing is on track

### Bidding Strategy
- [ ] Bid strategy matches objective
- [ ] Bid caps are reasonable
- [ ] Cost caps not limiting delivery
- [ ] Target CPA/ROAS based on historical data

## 7. Platform-Specific Checks

### Meta Ads
- [ ] Relevance score checked
- [ ] Delivery category reviewed
- [ ] Policy violations checked
- [ ] Pixel vs CAPI comparison
- [ ] Advantage+ creative options reviewed

### Google Ads
- [ ] Quality score analyzed
- [ ] Search terms reviewed (negatives needed?)
- [ ] Impression share analyzed
- [ ] Auction insights reviewed
- [ ] Display exclusions configured

### TikTok Ads
- [ ] Video completion rate monitored
- [ ] Creative diversity score
- [ ] Spark vs. paid ads compared
- [ ] Pangle placement reviewed

### LinkedIn Ads
- [ ] Audience size is 50K+ for broad
- [ ] Lead form is optimized
- [ ] Conversation ad strategy reviewed

## 8. Competitive Analysis

### Competitive Landscape
- [ ] Identified competitor ads
- [ ] Analyzed competitor creative angles
- [ ] Compared offers/messaging
- [ ] Identified differentiation opportunities

## 9. Action Items Prioritization

After completing audit, categorize findings:

**Critical (Fix Within 24 Hours)**
- Broken links/tracking
- Policy violations
- Budget not spending with no delivery

**High Priority (Fix This Week)**
- Underperforming ads to pause
- Budget reallocation needed
- Creative fatigue issues

**Medium Priority (Next Sprint)**
- A/B test opportunities
- Audience refinements
- Landing page optimizations

**Low Priority (Backlog)**
- New creative development
- Audience expansion testing
- New channel exploration
