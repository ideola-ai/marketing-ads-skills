# Twitter/X Content Guidelines

## Platform Overview
- **Character limit:** 280 characters per post
- **Thread length:** Unlimited (but 5-10 optimal)
- **Best times:** 8-10 AM, 12-1 PM, 5-6 PM
- **Vibe:** Conversational, witty, real-time

## Tweet Types

### Single Tweet
- **Length:** 200-240 characters optimal (leaves room for engagement)
- **Structure:** Hook + Value + Optional CTA
- **Best for:** Quick thoughts, news, hot takes

## Hashtag Strategy

### Twitter/X Hashtag Best Practices
- **Quantity:** 1-2 hashtags maximum per tweet
- **Relevance:** Must be highly relevant
- **Placement:** Within tweet or at end
- **Trending:** Jump on relevant trending topics

### Good Twitter/X Hashtags
- #marketing (broad)
- #ContentTwitter (niche community)
- #Indonesia (location)
- Trending topics when relevant

### Avoid
- More than 2 hashtags (looks spammy)
- Irrelevant trending hashtags
- Only hashtags (add value + hashtag)

## Twitter/X Algorithm Notes

### What Twitter/X Values
- **Engagement:** Replies, retweets, likes
- **Recency:** Very time-sensitive
- **Relevance:** To user's interests
- **Velocity:** Early engagement matters

### What Boosts Reach
- High engagement ratio (replies > likes)
- Retweets with comments
- Profile visits
- Follower growth from tweet
- Media (images, videos) perform better

### What Hurts Reach
- Low engagement
- Being flagged (misleading, spam)
- Tweeting at low-traffic times
- Only posting links

## Thread Writing Tips

### Technical Tips
- **Write as one document first** - Then break into tweets
- **Number your tweets** - Helps flow
- **End mid-sentence** - "Read more" effect
- **Connect tweets** - Use "..." or transition words

### Content Tips
- **Each tweet should stand alone** - If retweeted independently
- **Build momentum** - Save best for last
- **Make it scannable** - Use line breaks
- **Add visual to first tweet** - Boosts engagement

## Common Mistakes

1. **Wall of text** - Break into threads
2. **Weak hooks** - First tweet must grab attention
3. **Too many hashtags** - 1-2 max
4. **Posting links without context** - Add value
5. **Not engaging** - Reply to comments
6. **Inconsistent posting** - 3-5 tweets per day
7. **Self-promotion only** - Give more than you take
8. **Not using threads** - Best way to add value

## Twitter/X Content Ideas

1. **Thread breakdowns** - Explain complex topics
2. **Hot takes** - Share contrarian opinions
3. **Quick tips** - Bite-sized value
4. **Behind-the-scenes** - Work process, thoughts
5. **Curation** - Share best resources you found
6. **Live tweeting** - Events, conferences, while working
7. **Stories** - Personal experiences with lessons
8. **Frameworks** - Step-by-step approaches
9. **Before/After** - Transformations, results
10. **Questions** - Ask your audience for input
