# Content Format Specifications

## Static Posts (Single Image)

### Specifications
- **Instagram:** 1080x1080 (square) or 1080x1350 (portrait) or 1080x1920 (portrait 4:5)
- **LinkedIn:** 1200x1200 (square) or 1200x627 (landscape)
- **Twitter/X:** 1200x675 (landscape) or 1200x1200 (square)

### Copy Structure
```
[Hook - 1 line]
[Body - 2-3 sentences max]
[Value proposition]
[CTA]

[Hashtags]
```

### Visual Direction
- Single focal point
- Minimal text overlay (max 20% of image)
- Brand logo placement: top left or bottom right
- Leave space for caption overlay if needed

## Carousels (Multi-Slide)

### Specifications
- **Instagram:** 1080x1080 or 1080x1350 per slide
- **LinkedIn:** 1080x1080 or 1080x1350 per slide
- **Optimal slides:** 5-10 slides

### Visual Guidelines
- Consistent style across slides
- Progressive story flow
- Text hierarchy: headlines > subheads > body
- Use numbers or progress indicators

## Videos/Reels

### Specifications
- **Instagram Reels:** 9:16 aspect ratio, 1080x1920
- **TikTok:** 9:16 aspect ratio, 1080x1920
- **Optimal length:** 15-60 seconds

### Script Structure

**Hook (0-2 seconds)**
- Visual or audio hook immediately
- Bold statement or question
- "Wait for the end" type tease

**Body (2-45 seconds)**
- Deliver value quickly
- Use pattern interrupt every 3-5 seconds
- Show, don't just tell

**CTA (Last 5-10 seconds)**
- Clear action
- "Follow for more" or "Link in bio"
- Save/share prompt

### Caption Format
```
[Hook from video or related hook]

[Additional value or context]

[CTA]
[Engagement bait: "Which tip was most helpful?"]

[Hashtags - 3-5 blend of broad and niche]
```

### Visual Direction
- Fast cuts (every 1-3 seconds)
- Text overlays for key points
- Trending sounds when relevant
- Popping colors and contrast

## Stories

### Specifications
- **Instagram/TikTok:** 9:16, 1080x1920
- **Duration:** 15 seconds max per story
- **Optimal per session:** 3-5 stories

### Story Flow Formula

**Story 1: Hook/Tease**
- "Quick tip coming up"
- Question or poll
- Trending format

**Story 2-4: Content**
- Main value delivery
- Use interactive elements (polls, quizzes, sliders)
- Break into digestible chunks

**Last Story: CTA**
- "DM me [keyword]" or "Link in bio"
- "Add me for more content like this"

### Interactive Elements
- **Polls:** Engagement + quick feedback
- **Quiz:** Educational value
- **Slider:** Gauge interest/opinion
- **Question:** Collect responses
- **Countdown:** Build urgency

### Visual Guidelines
- Vertical text layout
- Large, readable fonts
- High contrast colors
- Minimal text per story

## Threads (Twitter/X, LinkedIn)

### Specifications
- **Twitter/X:** Up to thread-able posts, 280 chars each
- **LinkedIn:** Similar structure, more character space
- **Optimal length:** 5-10 posts in thread

### Thread Structure

**Post 1: Hook**
- Bold statement or insight
- "Thread ðŸ§µ" indicator
- Promise of value

**Post 2: Setup**
- Context or background
- Why this matters
- Personal connection (optional)

**Post 3-7: Main Content**
- One key point per post
- Number them or use emojis
- Build on previous point
- Actionable insights

**Post 8-9: Examples/Application**
- Real-world example
- Case study
- Before/after

**Last Post: CTA + Summary**
- Recap main point
- CTA (follow, retweet, link)
- "RT the first post to share"

### Visual Elements
- Add relevant image to first post
- Use emojis sparingly but effectively
- Bold key phrases with asterisks

### Copy Tips
- Complete sentences not required
- Break at logical pauses
- Each post should stand alone
- End mid-sentence to encourage "read more"
