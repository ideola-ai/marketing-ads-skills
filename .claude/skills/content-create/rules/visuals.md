# Visual Direction Guidelines

## Composition Principles

### Rule of Thirds
- Place key elements at intersection points
- Subject off-center creates dynamic interest
- Use grid overlay for precise placement

### Visual Hierarchy
```
1. Primary focal point (largest, brightest)
2. Secondary elements (supporting context)
3. Background (subtle, non-distracting)
```

### Negative Space
- Leave breathing room around key elements
- Don't overcrowd the frame
- White space = premium feel
- Minimum 20% negative space recommended

### Color Theory for Social

**High Contrast Colors (Stop the Scroll)**
- Yellow + Black
- Orange + Blue
- Pink + Green
- Red + White

**Trust Colors**
- Blue (professional, trustworthy)
- Green (growth, success)
- Purple (creative, premium)

**Action Colors**
- Red + Yellow (urgency, excitement)
- Orange (energy, enthusiasm)

## Typography Guidelines

### Font Pairing
```
Headline: Bold, display font
Body: Clean, readable font
Accent: Distinctive font for emphasis
```

### Size Guidelines
- Headline: 20-30% of image height
- Body text: Minimum 40px for readability
- CTA text: Slightly larger than body

### Readability Rules
- Max 3 font styles per image
- High contrast between text and background
- Avoid text over busy backgrounds
- Use overlays/blur for text readability

## Platform Visual Standards

### Instagram
- **Feed:** Aesthetic consistency matters
- **Grid:** Consider how posts look together
- **Reels:** Vertical, full-screen format
- **Stories:** Raw, authentic, ephemeral feel

**Visual Style Keywords:**
- Curated, aesthetic, lifestyle
- Warm tones, soft shadows
- Clean, minimal, or bold and colorful

### TikTok
- **Native:** Raw, authentic, unpolished
- **Trend:** Fast-paced, attention-grabbing
- **Format:** Vertical 9:16, full screen

**Visual Style Keywords:**
- Authentic, relatable, casual
- Bright, saturated colors
- Quick cuts, dynamic movement

### LinkedIn
- **Professional:** Clean, business-appropriate
- **Format:** Square or landscape preferred
- **Style:** Infographics, charts, professional photos

**Visual Style Keywords:**
- Professional, trustworthy, clean
- Blue/neutral color palette
- Data visualization, charts

### Twitter/X
- **Versatile:** Can be casual or professional
- **Format:** Landscape works best
- **Style:** News-style, meme-friendly

**Visual Style Keywords:**
- Bold, clear, shareable
- High contrast for timeline visibility
- Text-heavy graphics acceptable

## Brand Element Integration

### Logo Placement
- **Primary:** Top left corner
- **Secondary:** Bottom right corner
- **Size:** 10-15% of image width
- **Consistency:** Same position across all content

### Color Usage
- **60-30-10 Rule:**
  - 60% primary brand color
  - 30% secondary color
  - 10% accent color

### Pattern Usage
- Subtle patterns for texture
- Consistent patterns build recognition
- Don't let patterns compete with main content

---

## Format-Specific Visual Guidelines

### Static Posts
- Single clear focal point
- Minimal text overlay
- Brand elements visible but not dominant
- High-quality imagery (no pixelation)

### Carousels
- Consistent visual theme across slides
- Progressive storytelling
- Slide number indicators
- "Swipe" visual cues

### Video/Reels
- Fast cuts (every 1-3 seconds)
- Text overlays for key points
- Trending formats and sounds
- Popping colors and high contrast

### Stories
- Vertical text layout
- Large, readable fonts
- Interactive element placement
- Template-based for consistency

## Image Generation Prompts

When generating AI images, use this structure:

```
[Subject] [Action] [Setting] [Style] [Lighting] [Colors] [Camera]

Example:
"Professional woman working on laptop in modern cafe,
bright natural lighting, warm color palette,
shallow depth of field, canon 50mm, photorealistic,
instagram aesthetic, soft shadows"
```

**Professional:**
- Clean, minimal, corporate, modern, sleek

- Candid, natural, authentic, relatable, casual

**Luxury:**
- Elegant, premium, sophisticated, refined

**Playful:**

## Common Visual Mistakes to Avoid

1. **Too much text** - Keep text under 20% of image
2. **Low contrast** - Ensure text is readable
3. **Cluttered composition** - Use negative space
4. **Inconsistent branding** - Stick to brand guidelines
5. **Poor aspect ratio** - Use correct dimensions
6. **Low quality images** - High resolution only
7. **Hard-to-read fonts** - Clean, readable fonts only
