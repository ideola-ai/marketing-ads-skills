# Copywriting Frameworks for Social Media

## Hook Formulas

### Pattern Disruption
```
"Everything you know about [topic] is wrong."
"Stop [common action]. Try this instead."
"Unpopular opinion: [contrarian take]"
```

### Curiosity Gap
```
"The #1 mistake [audience] make..."
"Here's what nobody tells you about [topic]..."
"Why [result] is harder than you think..."
```

### Number Promise
```
"7 [benefits] that actually work"
"3 [things] I wish I knew earlier"
"The only [X] resources you'll ever need"
```

### Direct Address
```
"If you're [situation], read this."
"Attention [audience]..."
"This is your sign to [action]"
```

### Social Proof Hook
```
"How I got [result] in [timeframe]"
"The exact strategy that generated [X]"
"From [struggle] to [result] - here's how"
```

## CTA Templates

### Follow-Based
- "Follow for more [topic] tips"
- "Follow + turn on notifications"
- "Join [X] others who follow for [value]"

### Engagement-Based
- "Save this for later"
- "Share this with someone who needs it"
- "Tag someone who should see this"
- "Drop a [emoji] if you agree"

### Conversion-Based
- "Link in bio to [action]"
- "Click the link to [get benefit]"
- "DM me 'KEYWORD' for [resource]"

### Conversation-Based
- "What's your experience with [topic]?"
- "Which tip would you add?"
- "Agree or disagree? Let me know below"

## Storytelling Structures

### Before-After-Bridge
```
[Before: The struggle/pain point]
[After: The desired outcome]
[Bridge: How to get there - your solution]
```

### Problem-Agitate-Solve
```
[Problem: State the issue]
[Agitate: Make it feel personal/urgent]
[Solve: Present your solution]
```

### Star-Story-Solution
```
[Star: Introduce the character/yourself]
[Story: The journey/struggle]
[Solution: How you/they overcame it]
```

### AIDA (Attention-Interest-Desire-Action)
```
[Attention: Hook them]
[Interest: Build curiosity]
[Desire: Show benefit]
[Action: Clear CTA]
```

## Platform-Specific Tones

### Instagram
- **Casual but polished**
- Emoji-friendly (2-5 per post)
- Visual-first mindset
- Community-focused language
- Trend-aware but not forced

### TikTok
- **Raw and authentic**
- Conversational, like talking to a friend
- Fast-paced energy
- Trend-conscious
- Gen Z/Millennial friendly

**Example Tone:**
"Okay so I need to tell you guys about this thing that literally changed my life..."

### LinkedIn
- **Professional but human**
- Industry-relevant language
- Insight-driven
- Thought leadership tone
- Less emoji, more substance

**Example Tone:**
"In my 10 years in [industry], I've noticed one pattern that separates top performers..."

### Twitter/X
- **Conversational and witty**
- Concise and punchy
- Opinion welcome
- Thread-friendly
- Real-time responsive

**Example Tone:**
"Hot take: most [industry] advice is outdated. Here's what actually works in 2024..."


## Hashtag Strategy

### Hashtag Mix (for 20-30 hashtags)
- **5 broad:** #marketing #business #growth
- **10 niche:** #contentmarketing #smallbusinesstips
- **10 specific:** #indonesiabusiness #jakartastartup
- **5 branded:** #yourbrand #yourcampaign

### Hashtag Placement
- All at bottom (cleanest)
- First comment (keeps caption clean)
- Interspersed (for narrative posts)

### Hashtag Research
- Use competitor hashtags
- Check hashtag volume (aim for 10k-500k)
- Relevance over popularity
- Create hashtag groups by topic

## Copy Checklist

Before publishing, verify:
- [ ] Hook is in first 1-2 lines
- [ ] Value is clear and specific
- [ ] CTA is unambiguous
- [ ] Mobile-friendly formatting (line breaks)
- [ ] On-brand tone and voice
- [ ] Platform-appropriate length
- [ ] Hashtags are relevant
- [ ] No broken links
- [ ] Spelling and grammar checked
- [ ] Emoji usage fits platform norms
