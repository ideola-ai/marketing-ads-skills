# Ad Creative Strategy

## Creative Angles Framework

### The 7 Core Ad Angles

#### 1. Problem-Solution
```
Hook: "Struggling with [problem]?"
Body: Show the problem clearly
Solution: Present your product as the answer
CTA: "Fix it now"

Best for: Cold audiences, TOF
Format: Video, carousel
```

#### 2. Social Proof
```
Hook: "See what happened when..."
Body: Real results from real customers
Evidence: Screenshots, testimonials, numbers
CTA: "Get these results"

Best for: MOF, BOF
Format: Video, static image
```

#### 3. Educational
```
Hook: "Here's what nobody tells you about [topic]"
Body: Teach something valuable
Bridge: Connect to your product
CTA: "Learn more"

Best for: TOF, MOF
Format: Video, carousel
```

#### 4. Us vs Them
```
Hook: "Why [competitors] get it wrong"
Body: Show how you're different
Proof: Your unique advantage
CTA: "Choose the better option"

Best for: MOF, competitive markets
Format: Video, static
```

#### 5. Transformation
```
Hook: "From [before state] to [after state]"
Body: Show the journey
Proof: Real transformation
CTA: "Transform your [thing]"

Best for: All funnel stages
Format: Video, carousel
```

#### 6. Urgency/Scarcity
```
Hook: "Last chance to..."
Body: Limited time/quantity
Reason: Why the deadline
CTA: "Act now or miss out"

Best for: BOF, retargeting
Format: All formats
```

#### 7. Behind the Scenes
```
Hook: "How we make [product]"
Body: Show process, team, story
Human: The people behind it
CTA: "Support real people"

Best for: TOF, brand building
Format: Video, stories
```

## Ad Copy Formulas

### AIDA (Classic)
```
Attention: "Stop scrolling..."
Interest: "Here's why this matters..."
Desire: "Imagine if you could..."
Action: "Click here to..."

Works for: All funnel stages
```

### PAS (Problem-Agitate-Solve)
```
Problem: "Tired of [pain point]?"
Agitate: "It sucks when [consequence]"
Solve: "Here's the solution: [product]"
Action: "Try it now"

Works for: Cold audiences, TOF
```

### BAB (Before-After-Bridge)
```
Before: "Here's the problem you face"
After: "Here's what life could be like"
Bridge: "Here's how to get there"
Action: "Start your journey"

Works for: All funnel stages
```

### FAB (Features-Advantages-Benefits)
```
Feature: "Our product has [feature]"
Advantage: "This means you can [action]"
Benefit: "Which gives you [emotional benefit]"
Action: "Get this benefit today"

Works for: MOF, BOF
```

## Platform-Specific Creative Specs

### Meta Ads (Facebook + Instagram)

#### Feed Ads
- **Image:** 1080x1080 or 1080x1350
- **Video:** 4:5 or 1:1, up to 60 seconds
- **Text:** Primary text (125 chars), Headline (40 chars)
- **Link Description:** 30 chars

#### Stories Ads
- **Format:** 9:16 vertical
- **Duration:** 15 seconds max recommended
- **Max safe zone:** Center 60% (avoid UI)

#### Reels Ads
- **Format:** 9:16 vertical
- **Duration:** 15-30 seconds optimal
- **Hook:** Critical first 1-2 seconds

#### Carousel Ads
- **Cards:** 2-10 cards
- **Size:** 1080x1080 per card
- **Use:** Feature showcase, step-by-step

### TikTok Ads

#### In-Feed Ads
- **Format:** 9:16 vertical
- **Duration:** 15-60 seconds
- **Sound:** On by default (critical)
- **Hook:** First 1-2 seconds

#### Spark Ads
- **Format:** Boost organic content
- **Requirement:** Existing organic post
- **Benefit:** Native feel, better performance

### LinkedIn Ads

#### Sponsored Content
- **Image:** 1200x627 (landscape)
- **Carousel:** 1080x1080 per card
- **Video:** Up to 30 minutes (keep under 2 min)
- **Intro Text:** 150 chars optimal

#### Message Ads
- **Format:** Text-based with optional media
- **Subject Line:** 60 chars
- **Body:** 500-1000 chars

## Creative Testing Strategy

### Test Grid (What to Test)

| Test Variable | Options | Keep Constant |
|---------------|---------|---------------|
| Hook | 3-5 variations | Body, CTA, creative |
| Format | Video vs Image | Message, audience |
| Length | 15s vs 30s vs 60s | Content, hook |
| Style | UGC vs Professional | Message, format |
| CTA | 3-5 variations | Everything else |

### Testing Framework
```
Week 1: Launch with 5-7 creative angles
Week 2: Kill bottom 50%, scale top 20%
Week 3: Introduce 2-3 new variations of winners
Week 4: Final creative selections documented

Metrics to Watch:
- CTR (creative quality)
- CPC (hook effectiveness)
- Conversion rate (offer + creative match)
- CPA (overall efficiency)
```

### Creative Refresh Signals
```
Time to Refresh:
- Frequency > 4-5 per audience
- CTR declining week over week
- CPC increasing
- Engagement dropping

Refresh Actions:
- New hook, same body
- New creative, same hook
- New format entirely
- New offer
```

## UGC vs Professional Creative

### UGC (User-Generated Content)
```
✅ Advantages:
- Higher trust and authenticity
- Better performance on TikTok/Reels
- Lower production cost
- Native feel

❌ Disadvantages:
- Less brand control
- Variable quality
- May not fit all brands

Best For:
- TikTok, Instagram Reels
- Cold audiences
- Product demonstrations
- Testimonials
```

### Professional Creative
```
✅ Advantages:
- Complete brand control
- Consistent quality
- Polished, premium feel
- Scales better

❌ Disadvantages:
- Higher production cost
- May feel less authentic
- Lower engagement sometimes

Best For:
- LinkedIn, Facebook
- Brand awareness
- Product launches
- Luxury/high-ticket
```

## Creative Production Checklist

Pre-Production
- [ ] Creative angles defined
- [ ] Briefs written for each angle
- [ ] Format specs confirmed
- [ ] Brand guidelines reviewed

Production
- [ ] Hook iterations created (3-5 min)
- [ ] Multiple takes/versions shot
- [ ] Platform specs followed exactly
- [ ] Brand elements included properly

Post-Production
- [ ] Text overlays added
- [ ] Color grading done
- [ ] Sound/music mixed
- [ ] Platform export settings used

Quality Check
- [ ] Hook is in first 1-2 seconds
- [ ] CTA is clear and actionable
- [ ] Brand is visible but not dominant
- [ ] Mobile view tested
- [ ] Sound on/off tested
- [ ] All text is readable
