# Audience Segmentation & Targeting

## Audience Frameworks

### Cold Audience Targeting

#### Interest-Based Targeting
```
Primary Interests (Direct):
- [Industry] terms
- [Competitor] pages
- [Product category] interests

Secondary Interests (Related):
- [Related activities]
- [Lifestyle interests]
- [Problem-aware interests]

Budget: 50-60% of total
```

#### Lookalike Audiences (LAL)
```
Source: Customer list (email/phone)
- LAL 1%: Most similar (highest intent)
- LAL 2%: Broader reach
- LAL 5%: Scale audience

Source: Page engagers
- LAL 1%: Engaged lookalike
- LAL 2%: Interest expansion

Budget: 20-30% of total
```

#### Broad/Demographic Targeting
```
Demographics:
- Age: [Range based on customer data]
- Gender: [All or specific]
- Location: [Target geographies]
- Language: [Local language]

Behaviors:
- Device usage (mobile/desktop)
- Purchase behavior
- Life events

Budget: 10-20% of total
```

### Warm Audience Targeting

#### Page Engagers
```
Everyone who engaged with your page:
- Liked page
- Commented on posts
- Shared content
- Saved posts

Timeframe: Last 30-90 days
Budget: 40% of warm budget
```

#### Video Viewers
```
Based on video engagement:
- 25% of video watched (nurture)
- 50% of video watched (warm)
- 75%+ of video watched (hot)
- 95% of video watched (very hot)

Timeframe: Last 14-30 days
Budget: 30% of warm budget
```

#### Website Visitors
```
Pixel-based audiences:
- All visitors (last 30 days)
- Multiple sessions (visited 2+ times)
- Specific page visits (product page)
- Time on site (30+ seconds)

Timeframe: Last 30 days
Budget: 30% of warm budget
```

#### Instagram/FB Engagers
```
Cross-platform engagement:
- Engaged with any post (last 30 days)
- Sent message (last 30 days)
- Saved Instagram posts (last 30 days)

Timeframe: Last 30 days
Budget: Additional testing audience
```

### Hot Audience Targeting

#### Add to Cart
```
Users who added but didn't purchase:
- Exclude: Recent purchasers (7 days)
- Timeframe: Last 7 days
- Message: Complete your purchase
- Budget: 30% of hot budget
```

#### Initiated Checkout
```
Users who started checkout:
- Exclude: Recent purchasers (3 days)
- Timeframe: Last 7 days
- Message: Don't miss out
- Budget: 30% of hot budget
```

#### Product Page Viewers
```
High-intent page visitors:
- Exclude: Add to cart (7 days)
- Timeframe: Last 14 days
- Message: Still interested?
- Budget: 20% of hot budget
```

#### Customer List (Upsell)
```
Existing customers:
- Exclude: Recent purchasers (30-90 days)
- Source: CRM or purchase data
- Message: New product/complementary item
- Budget: 20% of hot budget
```

## Exclusion Strategies

### Purchase Exclusions
```
Exclude recent purchasers to prevent wasted spend:

Hot audiences: 3-7 days
Warm audiences: 14-30 days
Cold audiences: No exclusion needed
Upsell campaigns: 30-90 days
```

### Competitor Exclusions
```
For B2B or specific scenarios:
- Exclude competitor employees
- Exclude current customers
- Exclude unqualified leads
```

### Audience Fatigue Management
```
Rotate audiences to prevent ad fatigue:
- Exclude users who saw ad 5+ times
- Refresh creative weekly
- Rotate audiences every 2-3 weeks
```

## Audience Testing Strategy

### Test Priority Matrix
```
Priority 1 (Test First):
- Lookalike 1% from best customers
- Website visitors (last 30 days)
- Video viewers 50%+

Priority 2 (Test Second):
- Interest-based clusters
- Page engagers
- Lookalike 2%

Priority 3 (Test Later):
- Broad targeting
- Lookalike 5%
- Demographic-only
```

### A/B Testing Framework
```
Test Variable: Audience
Keep Constant: Creative, offer, landing page

Test Structure:
Ad Set A: Lookalike 1%
Ad Set B: Interest cluster 1
Ad Set C: Interest cluster 2

Winner Criteria:
- Lowest CPA
- Highest ROAS
- Statistical significance (95%+)
```

## Audience Insights

### Post-Campaign Analysis
```
Document findings after each campaign:

Winning Audiences:
- Which audiences converted best?
- What was the CPA/ROAS?
- What creative worked best?

Losing Audiences:
- Which audiences underperformed?
- What was the issue?
- Should we exclude or retest?

Insights:
- Demographic surprises
- Interest patterns
- Geographic patterns
- Device preferences
```

### Audience Expansion Signals
```
Green Light to Expand:
- CPA stable as budget increases 2x
- Multiple creatives working
- CTR above platform average
- Conversion rate steady

Red Light (Stop/Adjust):
- CPA rising with budget
- Single creative doing all the work
- CTR declining
- Frequency above 4-5
```

## Platform-Specific Targeting

### Meta Ads Targeting Options
```
Demographics:
- Age, gender, education, job title, relationship
- Life events (new job, moved, recently married)

Interests:
- Pages liked (2000+ categories available)
- Ads clicked on
- Device usage
- Purchase behaviors

Custom Audiences:
- Website traffic (pixel)
- App activity (SDK)
- Customer lists (email/phone)
- Engagement (page, video, form)

Lookalikes:
- Based on any custom audience
- 1% (top) to 10% (broad)
- Location-based availability
```

### TikTok Ads Targeting Options
```
Demographics:
- Age, gender, location, language
- Device type, OS version

Interests & Behaviors:
- Video interactions
- App install behaviors
- Creator interaction

Custom Audiences:
- Website visitors
- App users
- Engagement with TikTok content

Lookalikes:
- From customer lists
- From website visitors
```

### LinkedIn Ads Targeting Options
```
Professional Demographics:
- Job title, function, seniority
- Company size, industry, growth
- Skills, schools, degrees

Retargeting:
- Website visitors
- Lead form submissions
- Video viewers
- Event attendees

Account-Based Marketing:
- Upload company lists
- Match specific accounts
```
